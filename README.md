\# Real-Time Traffic Analytics Using Azure Stream Analytics



This repository contains the implementation of a real-time traffic analytics system using Azure Stream Analytics. The project includes:



\- Data ingestion from traffic sensors via MQTT

\- Data processing and streaming using Azure Stream Analytics and Kafka

\- Real-time dashboards and insights

\- Python scripts and notebooks for analytics and prediction



\## Milestones



\- \*\*Milestone 1\*\*: Setup and basic traffic data ingestion

\- \*\*Milestone 2\*\*: Data processing and ETL

\- \*\*Milestone 3\*\*: Streaming controllers and Telegram integration

\- \*\*Milestone 4\*\*: Analytics and prediction
